# Maps-app

## Descrição
  Este projeto foi desenvolvido para melhor compreender a utilização de mapas e buscas em um aplicativo react native por meio das APIs do google cloud.

## Intalação e execução
  Para instalar o projeto basta clonar o repositório via o comando
  ```git clone https://github.com/vomeajuda/maps-app.git```
  Ou baixar o zip do código fonte. Em seguida abra um terminal na raíz do projeto e execute o comando
  ```npm i```
  Logo após execute
  ```npx expo start```
  E por fim leia o qrcode gerado no terminal pelo app expo go em seu celular
